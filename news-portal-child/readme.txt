# News Portal Custom Child Theme
Visit <a href="https://github.com/DavidHalberstam/Sports_Broadcast_Journal">https://github.com/DavidHalberstam/Sports_Broadcast_Journal</a> for more details on editing the custom theme

## Core Frameworks:
- WordPress CMS v.4.9.5
- PHP 5.2.4 or Higher
- News Portal theme by Mystery Themes v.1.1.4
- Custom Child Theme News-Portal-Child

## Extension Frameworks:
- WP Customizer
- Foundation Template
- Walker Nav Menu
- FontAwesome icons library

- Perpetual Scroll
- Caching
- Slider
- Banner Ad Sections
- Masonry Grid
